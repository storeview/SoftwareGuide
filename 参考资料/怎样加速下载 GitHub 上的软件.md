# 怎样加速下载 GitHub 上的软件

## 现象

宽带看视频网站时不卡，但是 GitHub **下载软件资源时速度很慢**，甚至可能都没有 100KB/s

![image-20250502213126242](C:\Users\lilin\OneDrive\文档\03_GIT文件\SoftwareGuide\参考资料\assets\image-20250502213126242.png)

## 为什么会很慢🐌？
